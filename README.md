# Portal de Noticias - DigitalData

Este es un proyecto de un portal de noticias con panel de administración, cotización del dólar en tiempo real y modo oscuro / modo claro.

---

## Cómo Ejecutar el Proyecto

### Opción 1: GitHub Pages

Accede directamente desde:
https://kevdea247.github.io/Taller_De_InternetTPGrupal/

### Opción 2: Abrir directamente en el navegador

Para visualizar e interactuar con el portal de noticias localmente:

1. Descargue o clone la carpeta completa del proyecto.
2. Asegúrese de mantener la estructura de directorios intacta (css/, js/ y assets/).
3. Haga doble clic sobre el archivo **index.html** para abrir el portal en cualquier navegador web moderno (Chrome, Edge, Opera).
4. Ya puede realizar modificaciones locales al repositorio.

## Estructura del Proyecto

El proyecto está organizado en diferentes carpetas para facilitar su mantenimiento:

* css/: contiene los estilos visuales del sitio (colores, tipografías y diseño).
* js/: incluye la lógica y funcionalidades dinámicas (interacción, autenticación y gestión de noticias).
* assets/: almacena recursos multimedia como imágenes e íconos.
* index.html: punto de entrada principal del sitio.
* admin.html: vista correspondiente al panel de administración.
